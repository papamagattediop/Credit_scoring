3. RATIOS FINANCIERS MTIER

Justification thorique

Inspirs du modle Altman Z-Score et des pratiques bancaires :

  • DTI (Debt-to-Income) : Ratio crdit/revenu - indicateur cl de capacit de remboursement
  • Annuity Burden : Ratio annuit/revenu - charge mensuelle relative
  • Credit/Goods Ratio : Marge de financement (>1 = frais/intrts inclus)
  • Income per Family Member : Revenu disponible par personne
  • Employment Stability : Ratio anciennet emploi / ge

Ces ratios capturent la solidit financire du demandeur selon les standards bancaires.

def create_financial_ratios(df):
    """
    Cre des ratios financiers inspirs du Z-Score d'Altman et des pratiques bancaires.
    
    Rfrences:
    - Altman (1968) : Financial ratios for bankruptcy prediction
    - Basel III : Debt-to-Income requirements
    - Kaggle Top Solutions : Credit scoring best practices
    """
    print("Cration des ratios financiers...")
    
    # ========== RATIOS DE CAPACIT DE REMBOURSEMENT ==========
    
    # DTI (Debt-to-Income) - Ratio cl en scoring bancaire
    # Interprtation: <30% = bon, 30-40% = acceptable, >40% = risqu
    df['CREDIT_INCOME_RATIO'] = df['AMT_CREDIT'] / (df['AMT_INCOME_TOTAL'] + 1)
    
    # Annuity Burden - Charge mensuelle relative au revenu
    # Rgle bancaire: ne doit pas dpasser 33% du revenu mensuel
    df['ANNUITY_INCOME_RATIO'] = df['AMT_ANNUITY'] / (df['AMT_INCOME_TOTAL'] + 1)
    
    # Credit/Goods Ratio - Marge de financement
    # >1 signifie que le crdit inclut frais/intrts
    df['CREDIT_GOODS_RATIO'] = df['AMT_CREDIT'] / (df['AMT_GOODS_PRICE'] + 1)
    
    # Annuity/Credit Ratio - Vitesse de remboursement
    # Plus lev = remboursement plus rapide
    df['ANNUITY_CREDIT_RATIO'] = df['AMT_ANNUITY'] / (df['AMT_CREDIT'] + 1)
    
    # ========== RATIOS DE CAPACIT FAMILIALE ==========
    
    # Revenu par membre de famille
    df['INCOME_PER_FAMILY_MEMBER'] = df['AMT_INCOME_TOTAL'] / (df['CNT_FAM_MEMBERS'] + 1)
    
    # Crdit par membre de famille
    df['CREDIT_PER_FAMILY_MEMBER'] = df['AMT_CREDIT'] / (df['CNT_FAM_MEMBERS'] + 1)
    
    # Revenu par enfant (pour familles avec enfants)
    df['INCOME_PER_CHILD'] = df['AMT_INCOME_TOTAL'] / (df['CNT_CHILDREN'] + 1)
    
    # ========== RATIOS DE DURE ==========
    
    # Dure estime du crdit (en mois)
    df['CREDIT_TERM_MONTHS'] = df['AMT_CREDIT'] / (df['AMT_ANNUITY'] + 1)
    
    # Dure en annes
    df['CREDIT_TERM_YEARS'] = df['CREDIT_TERM_MONTHS'] / 12
    
    # ========== RATIOS DE GOODS ==========
    
    # Diffrence crdit - prix du bien (frais/intrts)
    df['CREDIT_GOODS_DIFF'] = df['AMT_CREDIT'] - df['AMT_GOODS_PRICE']
    
    # Pourcentage de frais
    df['FEES_PERCENTAGE'] = (df['AMT_CREDIT'] - df['AMT_GOODS_PRICE']) / (df['AMT_GOODS_PRICE'] + 1) * 100
    
    print(f"   {11} ratios financiers crs")
    return df

df = create_financial_ratios(df)

def create_temporal_features(df):
    """
    Cre des features temporelles drives des variables DAYS_*.
    
    Contexte:
    - Variables DAYS_* sont en jours ngatifs (avant la demande)
    - Conversion en annes pour interprtabilit
    - Ratios pour mesurer la stabilit relative
    """
    print("Cration des features temporelles...")
    
    # ========== CONVERSIONS EN ANNES ==========
    
    # ge en annes
    df['AGE_YEARS'] = -df['DAYS_BIRTH'] / 365
    
    # Anciennet emploi en annes
    # Note: valeurs positives aberrantes (365243) = retraits/sans emploi
    df['EMPLOYED_YEARS'] = -df['DAYS_EMPLOYED'] / 365
    df.loc[df['DAYS_EMPLOYED'] > 0, 'EMPLOYED_YEARS'] = 0  # Correction valeurs aberrantes
    
    # Anciennet enregistrement
    df['REGISTRATION_YEARS'] = -df['DAYS_REGISTRATION'] / 365
    
    # Anciennet pice d'identit
    df['ID_PUBLISH_YEARS'] = -df['DAYS_ID_PUBLISH'] / 365
    
    # Dernier changement de tlphone
    df['PHONE_CHANGE_YEARS'] = -df['DAYS_LAST_PHONE_CHANGE'] / 365
    
    # ========== RATIOS DE STABILIT ==========
    
    # Ratio emploi / ge (% de vie en emploi)
    # Interprtation: plus lev = carrire plus stable
    df['EMPLOYED_TO_AGE_RATIO'] = df['EMPLOYED_YEARS'] / (df['AGE_YEARS'] + 1)
    
    # Ratio enregistrement / ge
    df['REGISTRATION_TO_AGE_RATIO'] = df['REGISTRATION_YEARS'] / (df['AGE_YEARS'] + 1)
    
    # Ratio ID / ge
    df['ID_TO_AGE_RATIO'] = df['ID_PUBLISH_YEARS'] / (df['AGE_YEARS'] + 1)
    
    # ========== TRANCHES D'GE ==========
    
    # Catgories d'ge (utiles pour interactions)
    df['AGE_GROUP'] = pd.cut(df['AGE_YEARS'], 
                             bins=[0, 25, 35, 45, 55, 65, 100],
                             labels=['18-25', '26-35', '36-45', '46-55', '56-65', '65+'])
    
    # ========== INDICATEURS BINAIRES ==========
    
    # Client jeune (< 30 ans) - risque plus lev observ
    df['IS_YOUNG'] = (df['AGE_YEARS'] < 30).astype(int)
    
    # Client senior (> 55 ans) - risque plus faible observ
    df['IS_SENIOR'] = (df['AGE_YEARS'] > 55).astype(int)
    
    # Emploi rcent (< 1 an)
    df['IS_NEW_EMPLOYED'] = (df['EMPLOYED_YEARS'] < 1).astype(int)
    
    # Emploi stable (> 5 ans)
    df['IS_STABLE_EMPLOYED'] = (df['EMPLOYED_YEARS'] > 5).astype(int)
    
    # Flag anomalie emploi (retrait/sans emploi cod 365243)
    df['EMPLOYED_ANOMALY'] = (df['DAYS_EMPLOYED'] > 0).astype(int)
    
    print(f"   {15} features temporelles cres")
    return df

df = create_temporal_features(df)

4.5 TARGET ENCODING (K-Fold CV) NEW

Target Encoding - KazukiOnodera's Approach

Technique utilise par les top solutions Kaggle :

  • Encode les variables catgorielles par leur taux de dfaut moyen
  • Utilise K-Fold CV pour viter le data leakage
  • Plus informatif que One-Hot pour variables haute cardinalit

Variables encodes : NAME_CONTRACT_TYPE, NAME_INCOME_TYPE, NAME_EDUCATION_TYPE, NAME_FAMILY_STATUS, NAME_HOUSING_TYPE, OCCUPATION_TYPE, ORGANIZATION_TYPE, WEEKDAY_APPR_PROCESS_START

from sklearn.model_selection import StratifiedKFold

def target_encoding_kfold(df, target_col='TARGET', n_folds=5):
    """
    Target Encoding avec K-Fold Cross-Validation.
    
    Technique de KazukiOnodera pour viter le data leakage:
    - Pour chaque fold, encode avec la moyenne des autres folds
    - Le test set utilise la moyenne globale du train
    
    Source: https://github.com/KazukiOnodera/Home-Credit-Default-Risk/blob/master/py/003_TargetEncoding.py
    """
    print("Target Encoding avec K-Fold CV...")
    
    # Variables catgorielles  encoder
    cat_cols = [
        'NAME_CONTRACT_TYPE', 'NAME_TYPE_SUITE', 'NAME_INCOME_TYPE',
        'NAME_EDUCATION_TYPE', 'NAME_FAMILY_STATUS', 'NAME_HOUSING_TYPE',
        'OCCUPATION_TYPE', 'WEEKDAY_APPR_PROCESS_START', 'ORGANIZATION_TYPE',
        'FONDKAPREMONT_MODE', 'HOUSETYPE_MODE', 'WALLSMATERIAL_MODE'
    ]
    
    # Filtrer les colonnes existantes
    cat_cols = [c for c in cat_cols if c in df.columns]
    
    # Sparer train et test
    train_mask = df['is_train'] == 1
    
    for col in cat_cols:
        new_col = f'{col}_TE'  # Target Encoded
        df[new_col] = np.nan
        
        # Train: K-Fold encoding
        train_idx = df[train_mask].index
        train_data = df.loc[train_mask, [col, target_col]].copy()
        
        skf = StratifiedKFold(n_splits=n_folds, shuffle=True, random_state=42)
        
        for train_fold_idx, val_fold_idx in skf.split(train_data, train_data[target_col]):
            # Calculer la moyenne sur le fold d'entranement
            train_fold = train_data.iloc[train_fold_idx]
            encoding = train_fold.groupby(col)[target_col].mean()
            
            # Appliquer au fold de validation
            val_fold = train_data.iloc[val_fold_idx]
            df.loc[train_idx[val_fold_idx], new_col] = val_fold[col].map(encoding)
        
        # Test: utiliser la moyenne globale du train
        global_encoding = train_data.groupby(col)[target_col].mean()
        test_idx = df[~train_mask].index
        df.loc[test_idx, new_col] = df.loc[test_idx, col].map(global_encoding)
        
        # Remplir les NaN avec la moyenne globale
        global_mean = train_data[target_col].mean()
        df[new_col].fillna(global_mean, inplace=True)
    
    print(f"   {len(cat_cols)} variables encodes par Target Encoding")
    return df

df = target_encoding_kfold(df)

def aggregate_bureau_advanced(bureau, bureau_balance):
    """
    Agrgation avance de la table Bureau - inspire de KazukiOnodera.
    
    Amliorations par rapport  la version basique:
    1. Agrgations par fentre temporelle (all, 1y, 2y, 3y)
    2. Distribution des STATUS dans bureau_balance
    3. Features First/Last credit
    4. Ratios et diffrences avancs
    
    Source: https://github.com/KazukiOnodera/Home-Credit-Default-Risk/tree/master/py
    """
    print("Agrgation avance de la table Bureau...")
    
    # ========== DISTRIBUTION STATUS BUREAU_BALANCE ==========
    # Comptage de chaque statut (0, 1, 2, 3, 4, 5, C, X)
    
    bb_status = bureau_balance.groupby('SK_ID_BUREAU')['STATUS'].value_counts().unstack(fill_value=0)
    bb_status.columns = ['BB_STATUS_' + str(col) for col in bb_status.columns]
    
    # Statistiques temporelles
    bb_agg = bureau_balance.groupby('SK_ID_BUREAU').agg({
        'MONTHS_BALANCE': ['min', 'max', 'mean', 'size'],
    })
    bb_agg.columns = ['BB_MONTHS_MIN', 'BB_MONTHS_MAX', 'BB_MONTHS_MEAN', 'BB_COUNT']
    
    # Joindre status et stats
    bb_agg = bb_agg.join(bb_status)
    bb_agg = bb_agg.reset_index()
    
    # Joindre  bureau
    bureau = bureau.merge(bb_agg, on='SK_ID_BUREAU', how='left')
    
    del bb_agg, bb_status
    gc.collect()
    
    # ========== FONCTION D'AGRGATION HELPER ==========
    
    def agg_bureau_window(bureau_data, prefix=''):
        """Agrge bureau pour une fentre temporelle donne."""
        
        num_agg = {
            'DAYS_CREDIT': ['min', 'max', 'mean', 'std'],
            'DAYS_CREDIT_ENDDATE': ['min', 'max', 'mean'],
            'DAYS_CREDIT_UPDATE': ['min', 'max', 'mean'],
            'CREDIT_DAY_OVERDUE': ['max', 'mean', 'sum'],
            'AMT_CREDIT_MAX_OVERDUE': ['max', 'mean'],
            'AMT_CREDIT_SUM': ['sum', 'mean', 'max'],
            'AMT_CREDIT_SUM_DEBT': ['sum', 'mean', 'max'],
            'AMT_CREDIT_SUM_OVERDUE': ['sum', 'mean'],
            'AMT_CREDIT_SUM_LIMIT': ['sum', 'mean'],
            'AMT_ANNUITY': ['sum', 'mean', 'max'],
            'CNT_CREDIT_PROLONG': ['sum', 'mean'],
            'BB_MONTHS_MIN': ['min'],
            'BB_MONTHS_MAX': ['max'],
            'BB_COUNT': ['sum', 'mean'],
        }
        
        # Ajouter colonnes STATUS si elles existent
        status_cols = [c for c in bureau_data.columns if c.startswith('BB_STATUS_')]
        for col in status_cols:
            num_agg[col] = ['sum', 'mean']
        
        agg = bureau_data.groupby('SK_ID_CURR').agg(num_agg)
        agg.columns = [f'BUREAU{prefix}_' + '_'.join(col).upper() for col in agg.columns]
        
        # Nombre de crdits
        agg[f'BUREAU{prefix}_CREDIT_COUNT'] = bureau_data.groupby('SK_ID_CURR').size()
        
        return agg
    
    # ========== AGRGATIONS PAR FENTRE TEMPORELLE ==========
    
    # ALL TIME
    bureau_agg = agg_bureau_window(bureau, prefix='')
    
    # 1 YEAR (DAYS_CREDIT >= -365)
    bureau_1y = bureau[bureau['DAYS_CREDIT'] >= -365].copy()
    if len(bureau_1y) > 0:
        bureau_agg_1y = agg_bureau_window(bureau_1y, prefix='_1Y')
        bureau_agg = bureau_agg.join(bureau_agg_1y, how='left')
    
    # 2 YEARS (DAYS_CREDIT >= -730)
    bureau_2y = bureau[bureau['DAYS_CREDIT'] >= -730].copy()
    if len(bureau_2y) > 0:
        bureau_agg_2y = agg_bureau_window(bureau_2y, prefix='_2Y')
        bureau_agg = bureau_agg.join(bureau_agg_2y, how='left')
    
    # 3 YEARS (DAYS_CREDIT >= -1095)

def aggregate_previous_application(prev):
    """
    Agrge les donnes des demandes prcdentes chez Home Credit.
    
    Features cres:
    - Statistiques sur les demandes (count, montants)
    - Taux d'approbation/refus
    - Diffrences demand vs accord
    - Caractristiques des contrats
    """
    print("Agrgation de Previous Application...")
    
    # ========== PRTRAITEMENT ==========
    
    # Remplacer valeurs XNA/XAP
    prev = prev.replace({'XNA': np.nan, 'XAP': np.nan})
    
    # Crer features avant agrgation
    prev['APP_CREDIT_PERC'] = prev['AMT_APPLICATION'] / (prev['AMT_CREDIT'] + 1)
    prev['CREDIT_GOODS_DIFF'] = prev['AMT_CREDIT'] - prev['AMT_GOODS_PRICE']
    
    # ========== AGRGATION NUMRIQUE ==========
    
    num_agg = {
        'AMT_ANNUITY': ['sum', 'mean', 'max', 'min'],
        'AMT_APPLICATION': ['sum', 'mean', 'max', 'min'],
        'AMT_CREDIT': ['sum', 'mean', 'max', 'min'],

        'AMT_GOODS_PRICE': ['sum', 'mean', 'max'],

        'CNT_PAYMENT': ['sum', 'mean', 'max'],
        'APP_CREDIT_PERC': ['mean', 'max', 'min'],
        'CREDIT_GOODS_DIFF': ['mean', 'sum'],
    }
    
    prev_agg = prev.groupby('SK_ID_CURR').agg(num_agg)
    prev_agg.columns = ['PREV_' + '_'.join(col).upper() for col in prev_agg.columns]

    status_counts.columns = ['PREV_STATUS_' + col.upper().replace(' ', '_') for col in status_counts.columns]

    if 'PREV_STATUS_APPROVED' in prev_agg.columns:

    if 'PREV_STATUS_REFUSED' in prev_agg.columns:

            prev_agg['PREV_STATUS_REFUSED'] /

    contract_counts.columns = ['PREV_CONTRACT_' + col.upper().replace(' ', '_') for col in contract_counts.columns]
    prev_agg = prev_agg.join(contract_counts)
    
    # ========== FEATURES TEMPORELLES ==========
    
    # Demande la plus rcente
    prev_agg['PREV_DAYS_SINCE_LAST'] = prev.groupby('SK_ID_CURR')['DAYS_DECISION'].max()

    prev_agg['PREV_DAYS_SINCE_FIRST'] = prev.groupby('SK_ID_CURR')['DAYS_DECISION'].min()

def create_interaction_features(df):
    """
    Cre des features d'interaction et polynomiales.
    
    Focus sur:
    - EXT_SOURCE (top features)
    - Combinaisons crdit/revenu/ge
    """
    print("Cration des features d'interaction...")
    
    # ========== INTERACTIONS EXT_SOURCE ==========
    
    # Produits deux  deux
    df['EXT_SOURCE_1_2'] = df['EXT_SOURCE_1'] * df['EXT_SOURCE_2']
    df['EXT_SOURCE_1_3'] = df['EXT_SOURCE_1'] * df['EXT_SOURCE_3']
    df['EXT_SOURCE_2_3'] = df['EXT_SOURCE_2'] * df['EXT_SOURCE_3']
    
    # Produit des trois
    df['EXT_SOURCE_PROD'] = df['EXT_SOURCE_1'] * df['EXT_SOURCE_2'] * df['EXT_SOURCE_3']
    
    # Moyenne des EXT_SOURCE
    df['EXT_SOURCE_MEAN'] = df[['EXT_SOURCE_1', 'EXT_SOURCE_2', 'EXT_SOURCE_3']].mean(axis=1)
    
    # Moyenne pondre (EXT_SOURCE_3 a la plus forte corrlation)
    df['EXT_SOURCE_WEIGHTED'] = (
        df['EXT_SOURCE_1'] * 0.155 + 
        df['EXT_SOURCE_2'] * 0.160 + 
        df['EXT_SOURCE_3'] * 0.179
    ) / (0.155 + 0.160 + 0.179)
    
    # Min et Max
    df['EXT_SOURCE_MIN'] = df[['EXT_SOURCE_1', 'EXT_SOURCE_2', 'EXT_SOURCE_3']].min(axis=1)
    df['EXT_SOURCE_MAX'] = df[['EXT_SOURCE_1', 'EXT_SOURCE_2', 'EXT_SOURCE_3']].max(axis=1)
    
    # cart-type (variabilit des scores)
    df['EXT_SOURCE_STD'] = df[['EXT_SOURCE_1', 'EXT_SOURCE_2', 'EXT_SOURCE_3']].std(axis=1)
    
    # Nombre de sources disponibles
    df['EXT_SOURCE_COUNT'] = df[['EXT_SOURCE_1', 'EXT_SOURCE_2', 'EXT_SOURCE_3']].notna().sum(axis=1)
    
    # ========== INTERACTIONS CRDIT  GE ==========
    
    # Crdit par anne d'ge
    df['CREDIT_PER_AGE'] = df['AMT_CREDIT'] / (df['AGE_YEARS'] + 1)
    
    # Annuit par anne d'ge
    df['ANNUITY_PER_AGE'] = df['AMT_ANNUITY'] / (df['AGE_YEARS'] + 1)
    
    # Revenu par anne d'emploi
    df['INCOME_PER_EMPLOYED_YEAR'] = df['AMT_INCOME_TOTAL'] / (df['EMPLOYED_YEARS'] + 1)
    
    # ========== INTERACTIONS AVEC RGION ==========
    
    # Revenu relatif  la rgion
    df['INCOME_REGION_RATIO'] = df['AMT_INCOME_TOTAL'] * df['REGION_POPULATION_RELATIVE']
    
    # ========== FEATURES POLYNOMIALES EXT_SOURCE ==========
    
    # Carrs
    df['EXT_SOURCE_1_SQ'] = df['EXT_SOURCE_1'] ** 2
    df['EXT_SOURCE_2_SQ'] = df['EXT_SOURCE_2'] ** 2
    df['EXT_SOURCE_3_SQ'] = df['EXT_SOURCE_3'] ** 2
    
    # Cubes (non-linarit supplmentaire)
    df['EXT_SOURCE_1_CUB'] = df['EXT_SOURCE_1'] ** 3
    df['EXT_SOURCE_2_CUB'] = df['EXT_SOURCE_2'] ** 3
    df['EXT_SOURCE_3_CUB'] = df['EXT_SOURCE_3'] ** 3
    
    print(f"   {21} features d'interaction cres")
    return df
